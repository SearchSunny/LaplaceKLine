## Usage
#### Gradle

Project level build.gradle
```
allprojects {
    repositories {
        maven { url 'https://jitpack.io' }
    }
}
```
App level build.gradle
```
dependencies {
    implementation 'com.github.limxing:LaplaceKLine:1.1.4"'
}
```
